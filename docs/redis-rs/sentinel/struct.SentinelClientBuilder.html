Struct SentinelClientBuilder 

Source 
pub struct SentinelClientBuilder { /* private fields */ }
Available on crate feature sentinel only.
Expand description

Used to configure and build a SentinelClient. There are two connections that can be configured independently

  1. The connection towards the redis nodes (configured via set_client_to_redis_.. functions)
  2. The connection towards the sentinel nodes (configure via set_client_to_sentinel_.. functions)
